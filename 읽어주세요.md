# 🛫 webpilot 설치 안내 (동료용)

API 없는 사내 서비스(타임인아웃·비즈플레이·flow.team)를 브라우저 자동화로 감싼 도구예요.
로그인부터 조회·정리·상신까지 브라우저가 대신 해줍니다.

## 준비물
- **Mac** + **Google Chrome** (대부분 이미 있음 — 있으면 더 가볍게 설치됨)
- **Node.js** (없으면 → https://nodejs.org 에서 LTS 설치)

## 설치 (처음 1회)
1. `install.command` **더블클릭**
   - "확인되지 않은 개발자" 경고가 뜨면: **우클릭 → 열기**, 또는 시스템설정 → 개인정보 보호 및 보안에서 허용
   - 안 되면 터미널에서: `bash install.command`

## 실행
1. `start.command` **더블클릭** → 브라우저가 자동으로 열려요
2. 상단 **승객정보** 탭에서 본인 계정 등록
   - 타임인아웃(아이디/비번), 비즈플레이(아이디/비번), flow.team(API 키)
3. **목적지** 탭에서 원하는 자동화 실행
   - 초과근무 분석 · 출퇴근 정정 · 야근택시 결의 · 법인카드 미결의 조회

## 🔒 보안 (중요)
- 이 패키지엔 **아이디/비밀번호가 전혀 들어있지 않아요.** 각자 등록합니다.
- 등록한 로그인 정보는 **본인 맥에만 암호화 저장**(`.auth/accounts.enc`), 외부로 전송되지 않아요.
- 서버는 `127.0.0.1`(본인 컴퓨터)에서만 열려서 외부 접근 불가.

## 종료
- `start.command` 로 열린 **터미널 창을 닫으면** 종료돼요.
