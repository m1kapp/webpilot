# 🛫 webwing 설치 안내 (동료용)

API 없는 사내 서비스(타임인아웃·비즈플레이·flow.team)를 브라우저 자동화로 감싼 도구예요.
로그인부터 조회·정리·상신까지 브라우저가 대신 해줍니다. **맥·윈도우 둘 다 지원.**

## 준비물
- **Google Chrome** (대부분 이미 있음 — 있으면 훨씬 가볍게 설치됨)
- **Node.js** (없으면 → https://nodejs.org 에서 **LTS** 설치)

---

## 🍎 맥 (Mac)
1. **설치(1회)**: `install.command` 더블클릭
   - "확인되지 않은 개발자" 경고 → **우클릭 → 열기**, 또는 터미널에서 `bash install.command`
2. **실행**: `start.command` 더블클릭 → 브라우저 자동 오픈

## 🪟 윈도우 (Windows)
1. **설치(1회)**: `install.bat` 더블클릭
   - "Windows의 PC 보호" 경고 → **추가 정보 → 실행**
2. **실행**: `start.bat` 더블클릭 → 브라우저 자동 오픈

---

## 사용법 (공통)
1. 브라우저가 열리면 상단 **승객정보** 탭에서 본인 계정 등록
   - 타임인아웃(아이디/비번), 비즈플레이(아이디/비번), flow.team(API 키)
2. **목적지** 탭에서 원하는 자동화 실행
   - 초과근무 분석 · 출퇴근 정정 · 야근택시 결의 · 법인카드 미결의 조회
3. 종료: 실행 시 열린 **검은 창(터미널/명령프롬프트)을 닫으면** 종료

## 🔒 보안 (중요)
- 이 패키지엔 **아이디/비밀번호가 전혀 들어있지 않아요.** 각자 등록합니다.
- 등록한 로그인 정보는 **본인 컴퓨터에만 암호화 저장**(`.auth/accounts.enc`), 외부로 전송되지 않아요.
- 서버는 `127.0.0.1`(본인 컴퓨터)에서만 열려서 외부에서 접근 불가.
- 자동화는 **보이지 않는 백그라운드 크롬**으로 도는 거라, 쓰던 크롬 창은 안 건드려요.
